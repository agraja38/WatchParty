WatchParty GitHub

Teleparty-style sync, safer boundaries

Watch together from your own official streaming accounts.

WatchParty syncs play, pause, seek, chat, and a two-person WebRTC call without touching video streams or DRM-protected content.

View on GitHub

What it syncs

  • Play and pause
  • Seek and current time
  • Room chat
  • Two-person audio/video call

Features

Create a short room code, invite a friend, and keep official players in step. YouTube and generic HTML5 video support are fully implemented; major streaming platforms use safe best-effort HTML5 detection only.

Setup

  1. Download or clone the repository.
  2. Open your Chromium-based browser extension settings.
  3. Enable Developer Mode.
  4. Load the `extension/` folder unpacked.

Privacy Summary

WatchParty stores room state, chat, presence, and WebRTC signaling in Firebase. It does not store, relay, decrypt, or capture streaming video.
